Etiquetas HTML

¿Qué es HTML? Es el lenguaje que estructura el contenido de una página web. Permite organizar textos, imágenes, videos, secciones y más.

¿Para qué sirve? Sirve para definir la base de cualquier sitio web, indicando qué elementos aparecen y en qué orden.

¿Cómo se usa? Se utilizan etiquetas que van entre < >, cada una con una función específica. Por ejemplo: <p> para párrafos, <img> para imágenes, etc.

Conceptos de CSS

¿Qué es CSS? Es el lenguaje que da estilo a una página web.

¿Para qué sirve? Sirve para hacer que un sitio se vea atractivo.

Conceptos de JavaScript

¿Qué es JavaScript? Es el lenguaje que da interactividad y lógica a las páginas web. Permite que tu sitio “haga cosas”.

¿Para qué sirve? Sirve para crear acciones dinámicas como animaciones, validación de formularios, cálculos, menús interactivos y mucho más.

¿Cómo se usa? Se escribe dentro de <script> o en archivos externos .js, usando variables, condiciones, funciones y eventos.
